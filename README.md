# GL02 - ClearWater
A simple CLI-app for managing vCard contacts, and expor to JSON and CSV files.

## Installation
```bash
$ [sudo] npm install -g mocha
$ npm install
```

## Run tests
```bash
$ cd src
$ npm test
```

## Usage
```bash
$ npm start
```
